Calculo De Pulsaciones



¿Cómo se mide el pulso?

Para calcular el número de pulsaciones que debe tener una persona por cada 10 segundos de ejercicio aeróbico, la fórmula que se aplica es:

cuando el sexo es femenino: pulsaciones = (220 -edad) /10

cuando el sexo es masculino: pulsaciones = (210 -edad) /10
